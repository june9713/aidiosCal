# 엑셀 출력 기능 재정의 프로젝트 - 계획 0011

## 최종 목적
기존 간단한 엑셀 출력 기능을 사용자가 다양한 옵션을 선택할 수 있는 고급 엑셀 출력 기능으로 개선

## 새로운 요구사항
1. **엑셀 출력 버튼 클릭 시 설정 폼 표시**
   - 날짜 범위 설정 (기본: 지난 1개월 + 앞으로 6개월)
   - 날짜 조정 버튼: 1개월전, 1주전, 1개월후, 1주후
   - 개인일정 포함/제외 옵션
   - 다양한 탭별 파일 생성 옵션들

2. **다양한 출력 옵션**
   - 프로젝트별 탭을 포함한 별도의 파일 생성
   - 작성자별 탭을 포함한 별도의 파일 생성  
   - 월별 탭을 포함한 별도의 파일 생성
   - 주별 탭을 포함한 별도의 파일 생성
   - 우선순위별 탭을 포함한 별도의 파일 생성

3. **파일 생성 및 제공**
   - 여러 파일 생성 시 압축파일로 제공
   - HTML과 비슷한 스타일 유지

## 현재 상태 분석
- 기존 엑셀 출력 기능: `/schedules/export/excel` API 존재
- 프론트엔드: `exportToExcel()` 함수로 간단한 호출
- 백엔드: pandas, xlsxwriter 사용하여 단일 엑셀 파일 생성
- 우선순위별 색상 구분은 이미 구현됨

## 개발 단계
1. **프론트엔드 개선**
   - 기존 `exportToExcel()` 함수를 `showExcelExportModal()` 함수로 변경
   - 엑셀 출력 설정 모달 HTML 생성
   - 날짜 범위 설정 및 조정 버튼 기능
   - 다양한 출력 옵션 체크박스
   - 모달 CSS 스타일링

2. **백엔드 API 개선**
   - 기존 `/schedules/export/excel` API 확장
   - 새로운 파라미터 추가: 개인일정 포함여부, 출력 옵션들
   - 다중 탭 엑셀 파일 생성 로직
   - 여러 파일 생성 시 ZIP 압축 기능
   - HTML 스타일과 유사한 Excel 포매팅

3. **테스트 및 최적화**
   - 각 출력 옵션별 테스트
   - 대용량 데이터 처리 성능 확인
   - UI/UX 개선

## 기술적 고려사항
- ZIP 파일 생성을 위한 Python zipfile 모듈 사용
- 여러 워크시트를 가진 엑셀 파일 생성
- 메모리 효율적인 대용량 데이터 처리
- 다국어 파일명 지원
