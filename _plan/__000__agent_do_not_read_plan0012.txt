# Plan 0012: 월간 달력 weekday와 calendar-day 너비 일치 문제 해결

## 목표
월간 달력에서 weekday 헤더와 calendar-day 셀의 너비가 일치하지 않는 문제를 해결하여 완벽한 정렬을 달성

## 문제 분석
1. **현재 상황**:
   - `.calendar-weekdays`: `grid-template-columns: repeat(7, 1fr)` + gap 없음
   - `.calendar-days`: `grid-template-columns: repeat(7, 1fr)` + `gap: 1px`
   - 결과: weekday와 calendar-day의 너비가 맞지 않음

2. **원인**: 
   - gap 값의 차이로 인한 실제 컨텐츠 영역 너비 불일치
   - calendar-days는 gap으로 인해 실제 셀 너비가 줄어듦

## 해결 방안
1. **Gap 일치화**: calendar-weekdays에도 동일한 gap: 1px 적용
2. **배경색 구조 변경**: 
   - calendar-weekdays 배경을 #ddd로 설정 (gap 구분선 역할)
   - 개별 weekday에 #34495e 배경 적용
3. **Border 제거**: 불필요한 border-right 제거

## 실행 단계
1. ✅ CSS 수정: calendar-weekdays에 gap: 1px 추가
2. ✅ 배경색 구조 변경: weekday 개별 배경 적용
3. ✅ Border 스타일 정리

## 예상 결과
- weekday와 calendar-day의 완벽한 너비 일치
- 시각적으로 일관된 그리드 레이아웃
- 월간 달력의 전체적인 정렬 개선
