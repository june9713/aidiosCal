# aidiosCal 프로젝트 개선 계획 (Phase 7)

## 1. 알림 시스템 개선
### 1.1 알림 표시 기능
- 공유/메모 알림 표시 구현
- 일정 알림 시간 알림 구현
- 알림 목록 컴포넌트 개선

### 1.2 알림 관리 기능
- 알림 개별 확인(ack) 기능
- 알림 개별 삭제 기능
- 알림 전체 삭제 (2단계 확인)
- 알림 상태 관리 개선

## 2. 일정 표시 개선
### 2.1 정렬 및 표시
- 마감시간 기준 정렬 구현
- 제목과 내용 표시 최적화 (말줄임 처리)
- 완료 일정 표시 토글 기능

### 2.2 검색 기능
- 날짜 범위 검색
- 키워드 검색 (제목/내용/메모)
- 제외 키워드 검색
- 검색 UI 구현

## 3. 파일 업로드 기능
### 3.1 백엔드
- 파일 업로드 API 구현
- 파일 저장 및 관리 시스템
- 보안 및 용량 제한 설정

### 3.2 프론트엔드
- 파일 업로드 UI 구현
- 파일 미리보기 기능
- 업로드 진행률 표시

## 4. 사용자 기능
- 로그아웃 기능 구현
- 세션 관리 개선

## 구현 순서
1. 알림 시스템 개선 (1.1, 1.2)
2. 일정 표시 개선 (2.1)
3. 파일 업로드 기능 (3.1, 3.2)
4. 검색 기능 (2.2)
5. 사용자 기능 (4)

## 기술 스택
- Frontend: React, TypeScript
- Backend: FastAPI
- Database: PostgreSQL
- File Storage: Local/S3
