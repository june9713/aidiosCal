# 작업 기록 - 0009
## 작업 일시: 2024년 현재
## 작업 내용: 개인일정 추가 버튼 기능 구현

### 요청사항
사용자가 개인일정 추가 버튼을 누르면 개인일정 체크박스 `input type="checkbox"`를 체크한 뒤 일정 추가하는 것과 동일한 효과가 되도록 구현

### 수행 작업

1. **코드 분석 단계**
   - `static/js/main.js` 파일에서 개인일정 체크박스와 관련 버튼 구조 분석
   - 기존 폼 구조: 
     - 개인일정 체크박스: `id="schedule-individual"`
     - 개인일정 추가 버튼: `type="sub_pivate"` (기능 없음)
     - 공개일정 추가 버튼: `type="submit"`

2. **버튼 수정**
   - 파일: `static/js/main.js` 라인 1271 수정
   - 변경전: `<button type="sub_pivate">개인일정 추가</button>`
   - 변경후: `<button type="button" onclick="addPrivateSchedule()">개인일정 추가</button>`

3. **함수 구현**
   - 파일: `static/js/main.js`에 `addPrivateSchedule()` 함수 추가
   - 위치: `cancelAddSchedule()` 함수 다음
   - 기능:
     1. `schedule-individual` 체크박스를 체크
     2. `internal-add-schedule-form` 폼의 submit 이벤트 발생

### 구현된 기능
- 개인일정 추가 버튼 클릭 시 자동으로 개인일정 체크박스가 체크됨
- 체크박스 체크 후 폼이 자동으로 제출되어 일정이 추가됨
- 기존 공개일정 추가 기능은 변경 없음

### 기술적 세부사항
- JavaScript `document.getElementById()`로 체크박스 요소 획득
- `checked = true`로 체크박스 상태 변경
- `new Event('submit')`로 커스텀 submit 이벤트 생성
- `dispatchEvent()`로 폼 제출 이벤트 트리거

### 검증 방법
1. 일정 추가 폼 열기
2. 개인일정 추가 버튼 클릭
3. 개인일정 체크박스가 자동으로 체크되는지 확인
4. 폼이 자동으로 제출되어 개인일정이 추가되는지 확인

### 결과
개인일정 추가 버튼 기능이 요구사항에 따라 성공적으로 구현됨.
